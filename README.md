# AGV Path Planning using A* Algorithm

## Overview

This project simulates an Automated Guided Vehicle (AGV) navigating through a warehouse environment using the A* (A-Star) path planning algorithm.

The AGV finds the shortest path from a start location to a goal location while avoiding obstacles.

## Features

* A* Search Algorithm
* Obstacle Avoidance
* Shortest Path Calculation
* Grid-Based Environment
* Path Visualization using Matplotlib

## Technologies Used

* Python
* NumPy
* Matplotlib
* Heapq

## How It Works

1. A grid-based warehouse environment is created.
2. Obstacles are placed on the grid.
3. The AGV starts at a predefined location.
4. The A* algorithm evaluates possible paths.
5. The shortest obstacle-free path is selected.
6. The final path is displayed graphically.
